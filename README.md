# Customer Segmentation & Retention Analysis using RFM

## Overview

This project performs an end-to-end customer segmentation and retention analysis on the **Online Retail II dataset (2009–2011)** using Python.

The main goal of the project is to:

* Understand customer purchasing behaviour
* Identify high-value customers
* Detect customer churn risks
* Analyze monthly business trends
* Segment customers using RFM Analysis
* Generate actionable business insights

The project includes:

* Data Cleaning
* Exploratory Data Analysis (EDA)
* Revenue Analysis
* Average Order Value Analysis
* Customer Retention Analysis
* RFM Segmentation
* Business Recommendations
* Visualizations using Matplotlib and Seaborn

# 2. Exploratory Data Analysis (EDA)

Several business metrics were analyzed month-wise.

---

## Monthly Revenue Analysis

The monthly revenue trend was analyzed to identify seasonal business patterns.

### Key Findings

* Revenue peaks strongly during November
* Christmas season drives maximum sales
* April 2011 shows a noticeable dip
* December 2011 data is incomplete

### Business Insight

The business is highly seasonal and depends heavily on holiday demand.

---

## Average Order Value (AOV) Analysis

Average Order Value was calculated to understand spending behaviour.

### Formula

```python
AOV = Total Revenue / Number of Orders
```

### Key Findings

* January shows higher order values after Christmas
* March to July shows lower order values
* April 2011 had the lowest AOV
* November combines high order volume with high order value

### Business Insight

Customers spend significantly more during the festive season.

---

## New vs Returning Customer Analysis

Customers were classified into:

* New Customers
* Returning Customers

based on their first purchase month.

### Key Findings

* Returning customers steadily increased over time
* Returning customers greatly outnumber new customers by late 2011
* October shows small spikes in new customers before Christmas
* April 2011 shows weaker retention patterns

### Business Insight

The company relies heavily on customer retention and repeat purchases.

# 3. RFM Analysis

## What is RFM?

RFM is a customer segmentation technique based on:

| Metric    | Meaning                           |
| --------- | --------------------------------- |
| Recency   | How recently a customer purchased |
| Frequency | How often a customer purchases    |
| Monetary  | How much money a customer spends  |

---

## RFM Metrics Calculation

### Recency

Measures how recently the customer made a purchase.

```python
(reference_date - last_purchase_date).days
```

### Frequency

Counts total unique purchases.

```python
Number of unique invoices
```

### Monetary

Calculates total revenue generated by the customer.

```python
Sum of customer revenue
```

# 4. Customer Segmentation

Customers were grouped into business-focused segments.

## Segments

| Segment         | Description                               |
| --------------- | ----------------------------------------- |
| Champion        | Highly active and high spending customers |
| Loyal           | Frequent and consistent customers         |
| New/Promising   | Recently acquired customers               |
| At Risk         | Customers reducing activity               |
| Needs Attention | Moderately inactive customers             |
| Lost            | Inactive customers likely churned         |
